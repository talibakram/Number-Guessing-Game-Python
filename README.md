# Number Guessing Game 🎮

A fun and interactive terminal-based mini-game developed in Python. This project tests the user's guessing skills against the computer's randomly generated numbers, utilizing Python's built-in `random` module and loop control structures.

### 🚀 How to Play (Features):
* **Randomized Gameplay:** The computer intelligently selects a random number between 1 and 50 every time the game starts.
* **Limited Attempts:** To make it challenging, the user is given exactly 5 chances to guess the correct number.
* **Smart Feedback:** Provides dynamic prompts to the user for each attempt.
* **Win/Loss Logic:** Automatically detects if the user wins within the given chances. If all 5 attempts are exhausted, the game ends and reveals the hidden number.

### 🛠️ Tech Stack & Concepts Used:
* **Language:** Python 3
* **Core Modules:** `random` module for number generation.
* **Core Concepts:** `while`/`for` Loops, Conditional Statements (If-Else), Break statements, and User Input Handling.
